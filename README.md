# Dicord Log Bot
Records various logs that occur in the Discord Guild.

## Main function
* Save chat history
* Save attachment
* Save Server management history


## Detailed function
* Chat Log
* Chat Modify Log
* Guild Join Log
* Guild Leave Log
* Guild Ban Log
* Guild Channel Log
* Guild Emoji Log
* Guild Invite Log
* Guild Role Log
* Guild Voice Chat Log
* Typing Start Log
